# Exploratory Data Analysis (EDA) on Titanic Dataset

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset to uncover meaningful patterns, trends, and relationships among passenger attributes. The analysis includes data cleaning, statistical summaries, visualizations, and interpretation of key insights.

## Objective
To analyze the Titanic dataset and identify the factors that influenced passenger survival using statistical analysis and data visualization techniques.

## Dataset
- **Dataset:** Titanic Dataset
- **Format:** CSV
- **Source:** Kaggle

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- OpenPyXL
- Jupyter Notebook

## Project Workflow
1. Import required libraries
2. Load the dataset
3. Explore the dataset
4. Perform data cleaning
   - Handle missing values
   - Remove duplicates
   - Check data types
5. Generate statistical summaries
6. Create visualizations
7. Analyze correlations
8. Draw insights and conclusions

## Visualizations Included
- Survival Count
- Passenger Class Distribution
- Gender Distribution
- Age Distribution
- Fare Distribution
- Survival by Gender
- Survival by Passenger Class
- Age vs Fare Scatter Plot
- Embarked Distribution
- Correlation Heatmap

## Key Insights
- Most passengers traveled in Third Class.
- Female passengers had a higher survival rate than male passengers.
- First Class passengers had better survival chances.
- Most passengers were between 20 and 40 years old.
- Passenger class and gender showed a stronger relationship with survival than age.

## Outcome
This project demonstrates data exploration, statistical analysis, visualization, and analytical thinking skills using Python. It highlights how Exploratory Data Analysis (EDA) helps uncover meaningful insights that support data-driven decision-making.
